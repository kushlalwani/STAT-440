To simplify the equation for \(P(C=1|X_1,X_2,X_3)\), we need to rewrite Bayes equation with our specific variables plugged in. Since we set the prior to be 1/3 for all classes of wine, we can cancel out all of the prior terms in the equation.

\(P(C=1|X_1,X_2,X_3)=\frac{P(X_1,X_2,X_3|C=1)}{P(X_1,X_2,X_3|C=1) + P(X_1,X_2,X_3|C=2)+P(X_1,X_2,X_3|C=3)}\)

Since we are also assuming independence, we know that: \(P(X_1,X_2,X_3|C=j)=P(X_1|C=j)\times P(X_2|C=j)\times P(X_3|C=j)\)

So then we will have: \(P(C=1|X_1,X_2,X_3)=\frac{P(X_1|C=1)\times P(X_2|C=1)\times P(X_3|C=1)}{(P(X_1|C=1)\times P(X_2|C=1)\times P(X_3|C=1)) + (P(X_1|C=2)\times P(X_2|C=2)\times P(X_3|C=2))+(P(X_1|C=3)\times P(X_2|C=3)\times P(X_3|C=3))}\)

Since we are using a normal distribution for each variables marginal distribution, we can plug in the normal density function for each of the terms in the equation.

\(P(C=1|X_1,X_2,X_3)=\frac{\frac1{\hat{\sigma}_{1,1}\sqrt{2\pi}}e^{\frac{-(X_1-\hat{\mu}_1)^2}{2\hat{\sigma}_{1,1}}}\times \frac1{\hat{\sigma}_{2,1}\sqrt{2\pi}}e^{\frac{-(X_2-\hat{\mu}_2)^2}{2\hat{\sigma}_{2,1}}}\times \frac1{\hat{\sigma}_{3,1}\sqrt{2\pi}}e^{\frac{-(X_3-\hat{\mu}_3)^2}{2\hat{\sigma}_{3,1}}}}{(\frac1{\hat{\sigma}_{1,1}\sqrt{2\pi}}e^{\frac{-(X_1-\hat{\mu}_1)^2}{2\hat{\sigma}_{1,1}}}\times \frac1{\hat{\sigma}_{2,1}\sqrt{2\pi}}e^{\frac{-(X_2-\hat{\mu}_2)^2}{2\hat{\sigma}_{2,1}}}\times \frac1{\hat{\sigma}_{3,1}\sqrt{2\pi}}e^{\frac{-(X_3-\hat{\mu}_3)^2}{2\hat{\sigma}_{3,1}}}) + (\frac1{\hat{\sigma}_{1,2}\sqrt{2\pi}}e^{\frac{-(X_1-\hat{\mu}_1)^2}{2\hat{\sigma}_{1,2}}}\times \frac1{\hat{\sigma}_{2,2}\sqrt{2\pi}}e^{\frac{-(X_2-\hat{\mu}_2)^2}{2\hat{\sigma}_{2,2}}}\times \frac1{\hat{\sigma}_{3,2}\sqrt{2\pi}}e^{\frac{-(X_3-\hat{\mu}_3)^2}{2\hat{\sigma}_{3,2}}})+(\frac1{\hat{\sigma}_{1,3}\sqrt{2\pi}}e^{\frac{-(X_1-\hat{\mu}_1)^2}{2\hat{\sigma}_{1,3}}}\times \frac1{\hat{\sigma}_{2,3}\sqrt{2\pi}}e^{\frac{-(X_2-\hat{\mu}_2)^2}{2\hat{\sigma}_{2,3}}}\times \frac1{\hat{\sigma}_{3,3}\sqrt{2\pi}}e^{\frac{-(X_3-\hat{\mu}_3)^2}{2\hat{\sigma}_{3,3}}})}\)

This big ugly equation can be simplified significantly by turning it into a term that uses product and sum notation, we can also cancel out the \(\frac 1{\sqrt{2\pi}}\) term that appears.

\(P(C=1 | X_1, X_2, X_3) = \frac{\prod_{i=1}^{3} \frac{1}{\hat{\sigma}_{i,1}} e^{-\frac{(X_i -\hat{\mu}_i)^2}{2\hat{\sigma}_{i,1}^2}}}{\sum_{j=1}^{3} \prod_{i=1}^{3} \frac{1}{\hat{\sigma}_{i,j}} e^{-\frac{(X_i -\hat{\mu}_i)^2}{2\hat{\sigma}_{i,j}^2}}}\)

compute_class_probabilities <- function(X, mu, sigma) {
  # X: A vector of feature values
  # mu: matrix where each row corresponds to a class, and columns are means of features
  # sigma: matrix where each row corresponds to a class, and columns are SD of features
  
  numClasses <- nrow(mu)  # Number of classes
  numFeatures <- length(X) # Number of features
  
  likelihoods <- rep(0,numClasses)
  
  for (c in 1:numClasses) {
    means <- mu[c, ]
    sds <- sigma[c, ]
    
    likelihoods[c] <- prod(dnorm(X, mean = means, sd = sds))
  }
  
  # Compute posterior probabilities for each class
  posterior <- likelihoods / sum(likelihoods)
  
  return(posterior)
}

Exercise 2: Implement a Metropolis-Hastings Sampler

In the Bayesian normal model discussed in Week 08a notes, two facts are established in the case where \(X\sim N(\mu, \sigma^2)\): First, if \(\sigma^2\) is known, then a normal prior on \(\mu\) is a conjugate prior; second, if \(\mu\) is known, then an inverse gamma prior on \(\sigma^2\) is a conjugate prior.

However, neither of these cases is particularly realistic, since we usually dont know either \(\mu\) or \(\sigma\). This exercise will consider the case where we assume a priori that \(\mu\) and \(\sigma^2\) are independent with \(\mu\sim N(0,1)\) and \(\sigma^2\sim IG(1, 1)\). In other words, the joint prior is the product of the two marginal densities, normal for \(\mu\) and inverse gamma for \(\sigma^2\), so the prior is \[ \pi(\mu,\sigma^2) \propto \exp\left\{ -\frac{\mu^2}{2} \right\} \times (\sigma^2)^{-2} \exp\left\{ -\frac{1}{\sigma^2} \right\}. \] This prior on \((\mu,\sigma^2)\) is not a conjugate prior because in the posterior, \(\mu\) and \(\sigma^2\) are not independent. Nonetheless, this prior seems reasonable, and the goal of this exercise will be to produce approximate draws from the posterior.

Part (a) To keep things simple, assume that the data only consist of a single \(X\sim N(\mu, \sigma^2)\) rather than, say, an iid sample of size \(n\).

You know that the likelihood in this case is a normal density function of \(X\) with mean \(\mu\) and variance \(\sigma^2\). You also know that the posterior joint density for \((\mu,\sigma^2)\) is proportional to the likelihood times the prior. Based on these facts, supposing that we observe \(X=1.2\), write down a function that is proportional to the posterior.

Part (b) To implement a Metropolis-Hastings algorithm requires a proposal distribution. This is a distribution based on \((\mu_{\rm current}, \sigma^2_{\rm current})\) from which youll simulate \((\mu_{\rm proposed}, \sigma^2_{\rm proposed})\). In the Week 08b notes, this proposal density is denoted by \(Q\).

Again, to keep things simple, lets choose a proposal distribution that is symmetric, such as a bivariate normal distribution with zero correlation.

This symmetry means that \[ Q \left[ (\mu_c, \sigma^2_c) | (\mu_p, \sigma^2_p) \right] = Q \left[ (\mu_p, \sigma^2_p) | (\mu_c, \sigma^2_c) \right] \] where of course c means current and p means proposed. This means that the M-H acceptance probability (again, see the Week 08b notes) reduces to \[ A \left[ (\mu_c, \sigma^2_c) , (\mu_p, \sigma^2_p) \right] = \min \left\{ 1, \frac{f[(\mu_p, \sigma^2_p)]}{f[(\mu_c, \sigma^2_c)]}\right\}. \] Here, \(f\) is the posterior, because of the ratio of two \(f\) values, anything proportional to the posterior will work just as wellwhich is why the answer from part (a) can be used as \(f\).

Write code that implements 10,000 iterations as follows:

  1. Initialize \(\mu_c = 0\) and \(\sigma^2_c=1\).

  2. Define \(\mu_p = \mu_c + A\) and \(\sigma^2_p = \sigma^2_c + B\), where \(A\) and \(B\) are independent normal random variables with mean 0 and standard deviation 0.1.

  3. Calculate \(A\left[ (\mu_c, \sigma^2_c) , (\mu_p, \sigma^2_p) \right]\) as above.

  4. Generate standard uniform \(U\).

  5. If \(U<A\left[ (\mu_c, \sigma^2_c) , (\mu_p, \sigma^2_p) \right]\), replace \(\mu_c\) by \(\mu_p\) and \(\sigma^2_c\) by \(\sigma^2_p\).

  6. Record the values of \((\mu_c, \sigma^2_c)\), whether or not your replaced them with the new proposals in step 5.

  7. Return to step 2.

Part (c) Create a 2-dimensional scatterplot of the 10,000 recorded values of \((\mu_c, \sigma^2_c)\). This is your approximate sample from the posterior distribution.

Answer

Part (a)

Since the data is known to be a single point, we know that the likelihood function is the same as the mass function for the distribution.

\(L(X| \mu, \sigma^2) = \frac1{\sqrt{\sigma^2}}e^{-(X-\mu)^2/2\sigma^2}\)

We are also given that the marginal priors for the parameters are proportional to:

\(e^{-\frac{\mu^2}2} \times (\sigma^2)^{-2} e^{-\frac1{\sigma^2}}\)

Since we also know that the posterior distribution will be proportional to the likelihood times the prior.

\(\text p[(\mu,\sigma^2)|X] \propto e^{-\frac{\mu^2}2}(\sigma^2)^{-2} e^{-\frac1{\sigma^2}}\frac1{\sqrt{\sigma^2}}e^{-(X-\mu)^2/2\sigma^2}\)

After observing that X = 1.2, we get: \(e^{-\frac{\mu^2}2}(\sigma^2)^{-2} e^{-\frac1{\sigma^2}}\frac1{\sqrt{\sigma^2}}e^{-(1.2-\mu)^2/2\sigma^2}\)

Part (b)

We are going to use a proposal distribution that is symmetric and non-correlated. Using a bivariate normal distribution is a good suggestion for the proposal. Since there is no correlation and they are symmetric, this means that the transition probability of going from the current state to the proposed state is equal to the transition probability of going from the proposed state to the current state. This means that the transition terms in the acceptance probability will cancel out, reducing this just to a Metropolis sampler.

So our acceptance probability will be: \(A[ (\mu_c, \sigma^2_c) , (\mu_p, \sigma^2_p)] = \min(1, \frac{f[(\mu_p, \sigma^2_p)]}{f[(\mu_c, \sigma^2_c)]})\)

where \(f(\mu, \sigma^2)\) is our posterior distribution.

Now we will implement the sampler for 10 000 iterations

f <- function (mu, sigsq) {
  dnorm(mu) * 
  1/sigsq^2 * exp(-1/sigsq) * (sigsq > 0) *
  exp(-(1.2-mu)^2 / (2 * sigsq))
  }

# Step 1: Initialization (remember, C mean current and P means proposed)
muC <- 0 ; sigsqC <- 1
samples <- data.frame(mu = numeric(10000), sigsq = numeric(10000))

set.seed(440)



for (i in 1:10000) {
  # Step 2: Generate proposed values
  muP <- muC + rnorm(1)/10 
  sigsqP <- sigsqC + rnorm(1)/10
  
  # Step 3: Calculate the acceptance ratio
  A <- f(muP, sigsqP) / f(muC, sigsqC)
  
  # Steps 4 and 5: Generate U~unif(0,1) and compare with A
  
  if ( (U<-runif(1)) < A ) { 
    muC <- muP 
    sigsqC <- sigsqP
  }
  samples[i, ] <- c(muC, sigsqC)#store values of mu and sigsq current
}
